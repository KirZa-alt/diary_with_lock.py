# 🔐 Secret Diary App (Python + Streamlit)

A **secure and smart diary app** built with Python and Streamlit.
Write your private thoughts, protect them with a password, and keep everything **encrypted** 🔒

---

## ✨ Features

* 🔐 Password protected login
* 😳 Fake password mode (decoy diary)
* 🔒 Encrypted data storage (no readable files)
* 🚫 Auto lock after multiple wrong attempts
* ✍️ Add, view & delete diary entries
* 💻 Clean and simple UI

---

## ⚙️ Installation

```bash
pip install streamlit cryptography
```

---

## ▶️ Run the App

```bash
streamlit run diary_with_lock.py
```

---

## 🧠 How It Works

* Real password → opens your **private diary** 🔓
* Fake password → opens a **decoy diary** 🎭
* All entries are stored in **encrypted format** 🔒

---

## 🚀 Tech Stack

* Python
* Streamlit
* Cryptography
